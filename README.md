# ScoutIT.
Web Project- Shenkar


"ScoutIT." is an app for scouters / professional football people.
the app is for holding & managing data during football game (in realtime), in a visual and easy way for non-digital people. the system has been simplified for people in order to make the scouting job easy & fast as possible during live game.
